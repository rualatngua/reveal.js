Amazon EC2 - Elastic, Completely Controlled

  • Increase or decrease capacity within minutes, not hours or days.
  • Web service APIs, automatically scale itself up and down depending on its needs.
  • Root access to each one
  • Have access to console output of EC instances

Amazon EC2 - Flexible, Inexpensive

  • Multiple instance types, operating systems, and software packages.
  • Pay as you go
  • On-Demand Instances- pay for compute capacity by the hour with no long-term commitments
  • Reserved Instances - a low, one-time payment for each instance
  • Spot Instances - to bid on unused Amazon EC2 capacity and run those instances for as long as their bid exceeds the current Spot Price

Amazon EC2 - Reliable, Secure

  • The Amazon EC2 Service Level Agreement commitment is 99.95% availability for each Amazon EC2 Region.
  • APIs to configure firewall settings that control network access to and between groups of instances.
